# TweetPlay 🎥🐦

A full-stack platform combining video sharing and social interaction — inspired by YouTube and Twitter.

---

## 🚧 Live Demo

https://tweetxplay.vercel.app/

---

## ✨ Features

### 🎥 Video System

* Upload, update, delete videos
* Like videos
* Comment system

### 🐦 Tweet System

* Create, edit, delete tweets
* Like tweets

### 👤 User Features

* Authentication (JWT)
* Subscribe to channels
* Watch history tracking
* Subscriptions tracking

### 📂 Playlists

* Create playlists
* Add/remove videos

---

## 🛠️ Tech Stack

**Frontend**

* React.js

**Backend**

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/gauravkanwal/TweetPlay--Video_and_Social_Feed_Platform

### 2. Install dependencies

**Backend**
cd Backend
npm install

**Frontend**
cd Frontend
npm install

### 3. Create .env file in Backend

Add the following variables:

PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
CLOUDINARY_URL=your_cloudinary_config

### 4. Run the project

**Backend**
npm run dev

**Frontend**
npm start
